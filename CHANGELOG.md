# Changelog

## v1.0.4

- `install.sh` 安装完成后增加实际 Meta 内核版本输出，便于确认真实安装结果

- `uninstall.sh` 增加安装状态判断，避免未安装时输出不必要错误

## v1.0.3

- 精简 README，移除偏包装、偏宣传和非项目运行所需的描述

## v1.0.2

- 移除与项目运行无关的对外发布文案文件，保持仓库聚焦项目本身

- 优化 README 首页展示，新增徽章与更清晰的功能概览

- 新增 `repair.sh`，用于执行基础修复流程
