# OpenClash 短版发布文案

## 版本 1：简洁版

整理了一个 **OpenClash 一键安装 / 更新 / 卸载 / 修复脚本**，适用于 **OpenWrt / iStoreOS / ImmortalWrt**。

支持：
- 自动识别 `opkg` / `apk`
- 自动识别防火墙模式
- 自动匹配并安装 Meta 内核
- 菜单式管理

GitHub：
https://github.com/slobys/openclash-auto-installer

一键安装：
```sh
curl -fsSL https://raw.githubusercontent.com/slobys/openclash-auto-installer/main/install.sh | sh
```
